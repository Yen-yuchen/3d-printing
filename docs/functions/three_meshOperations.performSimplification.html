    Function performSimplification

    • performSimplification(
          state: ViewerState,
          elements: AppElements,
          targetType: "ratio" | "count",
          value: number,
      ): void

      Reduces the number of vertices in the model using the SimplifyModifier Can simplify by ratio (e.g., reduce to 50% of original) or by absolute vertex count Respects minimum vertex limits to prevent over-simplification

      Parameters

      • state: ViewerState

        The current state of the 3D viewer, which contains the active model (currentModel) to be modified.

      • elements: AppElements

        The DOM elements used to update the UI (e.g., status message and current vertex count label).

      • targetType: "ratio" | "count"

        The mode of simplification: "ratio" (percentage) or "count" (absolute number).

      • value: number

        The target numerical value. Represents a percentage (e.g., 0.5 for 50%) if targetType is "ratio", or an exact vertex count if "count".

      Returns void

      This function does not return a value. It modifies the 3D meshes in-place and updates the UI asynchronously.
